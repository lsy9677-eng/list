예선 대진표 링크 배포용 패키지

사용 방법:
1) 이 zip을 압축 해제합니다.
2) index.html, prelim_draw_01.png, prelim_draw_02.png 세 파일을 같은 폴더에 둡니다.
3) Cloudflare Pages / Firebase Hosting / Netlify / GitHub Pages 등에 폴더째 업로드합니다.
4) 배포 후 생성된 주소를 참가자에게 공유합니다.

권장:
- Cloudflare Pages를 쓰고 있다면 새 Pages 프로젝트로 이 폴더를 업로드하는 방식이 가장 간단합니다.
- 이미 운영 중인 230MATCH 앱에 넣을 경우 /draw/ 또는 /notice/ 같은 하위 폴더에 올리면 됩니다.
